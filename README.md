**1. Clone wallet sources**

```
git clone https://github.com/niobio-cash/niobio-wallet.git
```

**2. Install the dependencies**
- g++
- cmake
- boost
- qt5

On Ubuntu:
```
sudo apt-get install qt5-default cmake libboost-all-dev g++
```

**3. Build**

```
mkdir build && cd build && cmake .. && make
```
